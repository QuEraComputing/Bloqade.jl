# BloqadeExpr

Expressions and API definitions for Bloqade.

## Installation

<p>
BloqadeExpr is a &nbsp;
    <a href="https://julialang.org">
        <img src="https://raw.githubusercontent.com/JuliaLang/julia-logo-graphics/master/images/julia.ico" width="16em">
        Julia Language
    </a>
    &nbsp; package. To install BloqadeExpr,
    please <a href="https://docs.julialang.org/en/v1/manual/getting-started/">open
    Julia's interactive session (known as REPL)</a> and press <kbd>]</kbd> key in the REPL to use the package mode, and then type the following command:
</p>

For stable release:

```julia
pkg> add BloqadeExpr
```

For current master:

```julia
pkg> add BloqadeExpr#master
```
